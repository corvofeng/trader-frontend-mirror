# TradeForm 组件文档

`TradeForm` 组件提供了一个标准的表单界面，用于用户创建新的股票交易计划。

## 主要功能

- **基础交易信息**：支持输入证券代码、证券名称、买卖方向、目标价格和数量。
- **自动填充**：当用户从其他组件（如持仓列表）选中某只股票时，自动填充代码和当前市场价格。
- **价值估算**：根据输入的单价和数量，实时计算并展示预估的总交易金额。
- **立即执行选项**：支持勾选“立即执行”，直接发送到执行引擎，否则仅保存为待处理的计划。
- **备注支持**：允许为每笔交易计划添加文本说明。
- **个股配置入口**：点击“Configure Stock”可跳转至 `StockConfigEditor`，进行针对该股的个性化参数设置。

## 核心逻辑

- **状态管理**：使用 `useState` 追踪所有表单字段的变化。
- **账户绑定**：在提交时自动获取当前选中的账户别名，确保交易计划关联到正确的账户。
- **身份验证检查**：提交前检查用户登录状态。

## 属性 (Props)

| 属性名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `selectedStock` | `Stock | null` | 外部传入的预选股票对象 |
| `theme` | `Theme` | 当前主题 |
| `accountAlias` | `string` | 目标账户别名（可选） |

## 扩展建议

- **输入联想**：为证券代码输入框增加搜索建议和联想功能。
- **高级订单类型**：增加止损单（Stop Order）、限价止损单（Stop Limit）等复杂订单类型的支持。
- **资金预警**：在提交前，根据账户可用余额判断该计划是否可能导致保证金不足。
