# RiskAnalysis 组件文档

`RiskAnalysis` 组件提供高级的风险可视化工具，帮助交易员理解期权组合在不同市场场景下的表现。

## 主要功能

- **PnL 曲面图 (3D)**：
  - 使用 ECharts-GL 渲染。
  - 展示标的价格（S轴）、到期时间（T轴）与组合盈亏（Z轴）之间的三维关系。
  - 支持自动旋转和透视缩放，以便全方位观察风险区域。
- **保证金压力测试 (Margin Stress Test)**：
  - 模拟标的价格在极端波动（如 ±20%）下对账户保证金的影响。
  - 对比可用现金与保证金需求，自动识别爆仓风险并发出红色预警。
- **动态适配**：根据当前选中的账户和标的，实时获取并更新风险数据。

## 核心逻辑

- **三维曲面渲染**：将后端返回的 `payoff_matrix` 打平，映射到 ECharts 的 `surface` 系列中。
- **视觉映射**：通过 `visualMap` 根据 PnL 的正负自动渲染红色（亏损）和绿色（盈利）区域。
- **压力测试逻辑**：计算各场景下的 `net_margin`，并与 `available_cash` 进行对比，设置 `is_forced_close` 状态。

## 属性 (Props)

| 属性名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `theme` | `Theme` | 当前主题 |
| `selectedAccountId` | `string` | 当前选中的账户 ID |
| `selectedSymbol` | `string` | 当前选中的标的（可选，用于过滤） |

## 扩展建议

- **希腊字母曲面**：除了 PnL，还可以提供 Delta 曲面、Gamma 曲面等可视化选项。
- **多标的组合风险**：目前主要针对单一标的，未来可以扩展到整个账户的综合风险分析。
- **交互式压力测试**：允许用户手动输入自定义的价格波动率，观察即时的保证金变化。
