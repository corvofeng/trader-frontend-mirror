# StrategyCreator 组件文档

`StrategyCreator` 是一个交互式的工具，允许用户构建、模拟并保存自定义的期权交易策略。

## 主要功能

- **模板引导**：提供常用策略模板（看涨/看跌价差、铁鹰、跨式等），点击即可快速填充参数。
- **自定义配置**：支持手动添加任意数量的期权腿（Legs），自定义行权价、到期日、操作方向（买入/卖出）和数量。
- **风险与分类**：为策略设置分类（Bullish, Bearish, Neutral, Volatility）和风险等级（Low, Medium, High）。
- **实时模拟**：计算策略的理论总成本、当前价值以及预期盈亏。
- **保存管理**：将构建好的策略保存到服务器，以便后续在持仓管理或交易计划中使用。

## 核心逻辑

- **模板映射 (`STRATEGY_TEMPLATES`)**：预定义了一系列经典策略的结构，通过 `handleTemplateSelect` 进行实例化。
- **动态表单**：使用局部状态管理 `positions` 数组，支持实时增删改。
- **成本计算**：自动根据买入/卖出方向计算净支出（Debit）或净收入（Credit）。

## 属性 (Props)

| 属性名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `theme` | `Theme` | 当前主题 |
| `selectedSymbol` | `string` | 当前选中的标的代码 |
| `onStrategyCreated` | `(strategy) => void` | 策略创建成功后的回调 |

## 扩展建议

- **盈亏图预览**：在创建过程中实时展示该策略的到期盈亏曲线图。
- **期权链联动**：允许用户直接从 `OptionsChain` 中点击某个合约来将其加入到当前构建的策略中。
- **保证金计算**：增加对策略所需保证金（Margin Requirement）的初步估算。
