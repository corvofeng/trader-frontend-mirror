# OptionsTradePlans 组件文档

`OptionsTradePlans` 组件用于管理和生成自动化的期权交易计划，目前主要专注于比率价差（Ratio Spread）策略。

## 主要功能

- **计划列表**：展示由后端生成的比率价差计划，按到期日分组。
- **智能筛选与排序**：支持按期权类型（Call/Put）、值度（ITM/ATM/OTM）、保存状态进行过滤，并按杠杆、净权利金等指标排序。
- **动作推荐**：自动标识计划的建议动作（Open: 开仓, Close: 平仓, Hold: 继续持有）。
- **计划保存与同步**：允许用户将生成的计划保存到个人库，并支持手动刷新已保存计划的最新数据。
- **详细分析**：提供每个计划的买入/卖出腿详情、净权利金计算、杠杆比例以及逻辑说明。
- **在线编辑**：内置编辑模态框，支持修改计划的标签、行权价、目标价差等核心参数。

## 核心逻辑

- **值度判断 (`getMoneyness`)**：根据标的价格与买入腿行权价的偏离度，实时判断计划的 ITM/ATM/OTM 状态。
- **数据刷新**：通过 `refreshRatioSpreadPlan` 接口，在不重新生成整个列表的情况下，更新特定计划的实时行情。
- **颜色编码**：使用不同的边框和背景色（蓝色：ATM, 绿色：ITM, 黄色：OTM）来区分计划的风险/收益特征。

## 属性 (Props)

| 属性名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `theme` | `Theme` | 当前主题 |
| `selectedSymbol` | `string` | 当前选中的标的代码 |
| `selectedAccountId` | `string` | 当前选中的账户 ID |
| `userId` | `string` | 当前用户 ID |

## 扩展建议

- **更多策略类型**：支持除比率价差外的其他复杂策略计划，如铁鹰、日历价差等。
- **自动化执行对接**：增加“一键执行”按钮，将计划直接发送到交易执行引擎。
- **历史回测集成**：在计划详情中展示该类型计划在历史数据中的表现评估。
