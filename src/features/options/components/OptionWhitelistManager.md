# OptionWhitelistManager 组件文档

`OptionWhitelistManager` 组件用于管理期权合约白名单，通常用于限制交易品种或标记需要重点关注的合约。

## 主要功能

- **白名单展示**：按到期月份分组展示已加入白名单的期权合约。
- **快速添加**：通过合约代码快速添加新项，支持失焦后自动获取合约详情（如行权价、到期日等）。
- **字段编辑**：支持修改现有白名单项的持有类型（Right/Obligation/Covered）、数量、原因和备注。
- **删除管理**：一键从白名单中移除不再需要的合约。
- **状态同步**：支持手动刷新，确保展示的数据与后端同步。

## 核心逻辑

- **自动填充 (`handleContractCodeBlur`)**：当用户输入合约代码并失焦时，调用 `getOptionContractDetail` 接口预填充表单，减少手动输入量。
- **数据分组**：通过 `reduce` 函数将白名单列表按 `expiry_month` 进行聚合展示，方便按月度管理。
- **内联编辑**：在表格行内直接切换编辑状态，提供更流畅的交互体验。

## 属性 (Props)

| 属性名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `theme` | `string` | 当前主题 |
| `userId` | `string` | 用户 ID |
| `accountId` | `string` | 账户 ID（可选） |

## 扩展建议

- **批量导入**：支持从 CSV 或剪贴板批量导入合约代码。
- **价格提醒集成**：为白名单内的合约设置特定的价格或隐波提醒。
- **关联持仓**：在持仓列表中高亮显示属于白名单的合约，方便识别。
