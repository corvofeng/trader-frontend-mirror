# OptionsPortfolio 组件文档

`OptionsPortfolio` 组件用于管理和展示用户的期权持仓情况，支持多种视图模式和实时价格更新。

## 主要功能

- **多视图模式**：
  - `expiry`: 按到期日分组展示。
  - `strategy`: 按策略（如垂直价差、跨式等）分组展示。
  - `symbol`: 按标的代码分组展示。
- **实时价格监控**：通过 WebSocket 订阅最新价格，并自动计算各持仓的实时盈亏、希腊字母和隐含波动率。
- **策略推导**：能够根据选中的多个期权腿，自动推导可能的策略名称（如 "Bull Call Spread"）。
- **组合管理**：支持将多个单腿持仓“打包”保存为复杂策略，或将现有策略“拆解”。
- **活动日志**：记录持仓的打开、平仓和数量变更，并提供历史日志面板。
- **快捷操作**：支持一键平仓、快速滚动到特定到期日（ExpiryFastNav）。

## 核心逻辑

- **数据同步**：集成了 WebSocket (`useOptionPriceWebSocket`) 和定时轮询，确保持仓数据和行情数据的准确性。
- **差异对比 (`processDiff`)**：在数据更新时对比前后持仓差异，自动生成活动日志并触发通知弹窗。
- **持久化状态**：展开的分组状态和滚动位置通过 Cookie 持久化，提升用户返回页面时的体验。

## 属性 (Props)

| 属性名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `theme` | `Theme` | 当前主题 |
| `selectedAccountId` | `string` | 当前选中的账户 ID |
| `refreshKey` | `number` | 用于强制触发数据刷新的 Key |
| `optionsData` | `OptionsData` | 期权市场数据（可选） |
| `selectedSymbol` | `string` | 当前选中的标的（可选，用于过滤） |

## 扩展建议

- **损益分析图**：在策略分组下直接展示该组合的盈亏图。
- **批量操作**：增加批量平仓、批量调整止损止盈的功能。
- **希腊字母汇总**：在页面顶部展示整个投资组合的汇总 Delta, Gamma, Theta。
