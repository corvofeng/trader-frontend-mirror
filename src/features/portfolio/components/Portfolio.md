# Portfolio 组件文档

`Portfolio` 组件是交易日志页面的核心，负责全面展示用户的股票持仓、成交记录、资产走势以及组合分析。

## 主要功能

- **资产概览 (StatsGrid)**：实时展示总市值、今日盈亏、总仓位占比等核心指标。
- **资产走势 (PortfolioTrend)**：展示选定日期范围内的账户总资产变化趋势。
- **持仓热力图 (PortfolioHeatmap)**：根据持仓市值和盈亏情况，直观展示各股票的仓位分布。
- **持仓列表 (HoldingsTable)**：支持分页、排序和搜索，提供针对个股的深入分析入口（StockAnalysisModal）。
- **成交记录 (TradesTable)**：展示近期的交易活动。
- **资产配置 (Pie Chart)**：直观展示不同股票在组合中的市值占比。
- **组合分析 (PortfolioAnalysisPanel)**：集成高级分析功能，如相关性分析、风险评估等。
- **分享与导出**：支持生成持仓截图预览，并下载为图片，方便分享。支持共享视图模式（Read-only）。

## 核心逻辑

- **多模式适配**：支持实时视图、快照视图和共享视图。
- **数据获取**：根据当前选中的账户和日期范围，调用 `portfolioService` 获取持仓、交易和趋势数据。
- **响应式设计**：针对移动端和桌面端优化了布局（如 Pie Chart 的位置和比例）。
- **截图功能**：利用 `html-to-image` 将组件渲染为高分辨率图片。

## 属性 (Props)

| 属性名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `holdings` | `Holding[]` | 持仓数据列表 |
| `theme` | `Theme` | 当前主题 |
| `dateRange` | `{startDate, endDate}` | 当前选择的日期范围 |
| `onDateRangeChange` | `(range) => void` | 日期范围变更回调 |
| `selectedAccountId` | `string` | 当前选中的账户 ID |
| `isSharedView` | `boolean` | 是否为分享视图（只读） |

## 扩展建议

- **多账户汇总**：增加一键汇总所有账户资产的功能。
- **导出 CSV**：除了图片导出，增加持仓和交易记录的 CSV 导出功能。
- **基准对比**：在资产走势图中增加与大盘（如 S&P 500 或沪深 300）的对比曲线。
